Make This In An Hour

  1. Go to namecheap.com and register a domain. This is optional and only if you want a custom domain name.
  2. Make a Github account
  3. Download Git.
  4. Create a directory to hold your website and cd into it. mkdir my-site && cd my-site. All future commands will assume you are within this directory.
  5. Make a new Github repository for your site titled .github.io. For example, my Github account is smilli, so my repository is named smilli.github.io.
  6. While in the directory for your website run git init and git remote add origin https://github.com/username/username.github.io.git with username replaced by your username.
  7. Make a file called "index.html" in your website's folder with the HTML for your main page. Don't spend long on this; it can be as simple as the following:
    8. <html>
<head>
  <title>YOUR SITE TITLE</title>
  <link rel="stylesheet"
  href="https://maxcdn.bootstrapcdn.com/bootstrap/3.3.4/css/bootstrap.min.css">
</head>
<body>
  <h1>YOUR SITE TITLE</h1>
  <p>Your text blahblahbhalbha</p>
  <p>Another paragraph!  Maybe more text!</p>
</body>
</html>
  8. Enter whatever text or html you want in the file. You can view the changes you're making locally by running python -m SimpleHTTPServer 8000 and then navigating to http://localhost:8000 in your browser.
  9. Now run the following commands: 
    git add .
git commit -m "Initial commit"
git push -u origin master
  10. Go to http://yourusername.github.io. You should now see your site! Congratulations, you put something online.
  11. If you want to add your own domain name, create a file called "CNAME" in all caps. Then add a single line that says 'www.yourdomainname.com' to the file. For example the CNAME file for this website looks like this: 
    www.makethisinanhour.com
    View details on CNAME configuration here.
  12. Like before, push your changes to Github. 
    git add .
git commit -m "Add CNAME"
git push
  13. Now login to http://namecheap.com and click on the domain you want under the list of your domains. The list of domains should look like this:

    After clicking on your domain, you should see a link that says "All Host Records" on the left sidebar underneath "Host Management". Click this. Now you should see the default records that Namecheap adds for your site.
